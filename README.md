# 🔄 URL to Markdown Converter

Convert any web page to clean, AI-ready Markdown using Claude AI. **Completely free!**

Created by **Bilal Dalgün**

---

## ✨ Features

- ✅ Convert URLs to clean Markdown
- ✅ Batch processing (multiple URLs at once)
- ✅ Conversion history (saved locally)
- ✅ Edit markdown before saving
- ✅ Download as .md file
- ✅ **ZERO cost** - Uses your Claude subscription

## 🚀 How to Use

### Step 1: Copy the Code

Copy the `artifact.jsx` file from this repository

### Step 2: Go to Claude.ai

Visit https://claude.ai (you need a **paid Claude account**)

### Step 3: Create New Artifact

Click the **"+"** button or **"Create artifact"** button

### Step 4: Paste the Code

Paste the artifact.jsx code into Claude

### Step 5: Use It!

1. Enter a URL: `https://example.com`
2. Click **"✨ Send to Claude"**
3. Get your markdown in the chat!

That's it! 🎉
