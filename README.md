## Nasıl Kullanılır?

### Tek URL
1. **Tek URL** modunda URL girin
2. **"Prompt Oluştur ve Kopyala"** butonuna basın
3. Claude chat'e yapıştırın → markdown çıktısını alın

### Toplu URL + ZIP İndirme
1. **Manuel Liste** moduna geçin
2. Her satıra bir URL yazın
3. **"Prompt Oluştur ve Kopyala"** butonuna basın
4. Claude chat'e yapıştırın — Claude her sayfayı `---` ile ayırarak verir
5. **"Markdown Yapıştır"** butonuna basın → Claude'un tüm yanıtını yapıştırın → Kaydet
6. Araç her sayfayı otomatik ayırır ve geçmişe kaydeder
7. **"ZIP İndir"** → Her sayfa ayrı `.md` dosyası olarak zip'e paketlenir

## Özellikler

- **Tek URL modu** — tek sayfa
- **Manuel Liste modu** — birden fazla URL, satır satır
- **Otomatik kopyalama** — butona basınca prompt hazır
- **Markdown Yapıştır** — Claude çıktısını araca yükle
- **Otomatik bölme** — çoklu URL yanıtını ayrı sayfalara böler
- **ZIP İndir** — her sayfa ayrı `.md` dosyası olarak
- **Markdown çıktı paneli** — token / karakter / satır istatistikleri
- **Kopyala / İndir .md / Düzenle**
- **Geçmiş** — son 20 dönüşüm tarayıcıda saklanır
