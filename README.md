## Nasıl Kullanılır?

### Tek Sayfa
1. **Tek URL** modunda URL girin
2. **"Prompt Oluştur ve Kopyala"** butonuna basın
3. Claude chat'e yapıştırın → markdown çıktısını alın
4. **"Markdown Yapıştır"** → Claude'un yanıtını yapıştırın → Kaydet
5. **"İndir .md"** ile indirin

### Toplu Sayfa + ZIP
1. **Manuel Liste** moduna geçin, her satıra bir URL yazın
2. **"Prompt Oluştur ve Kopyala"** → Claude'a yapıştırın
3. Claude her sayfayı `---` ayırıcıyla tek yanıtta verir
4. **"Markdown Yapıştır"** → tüm yanıtı yapıştırın → Kaydet
5. Araç sayfaları otomatik ayırır, geçmişe kaydeder
6. **"ZIP İndir"** → her sayfa ayrı `.md` dosyası olarak indirilir

## Özellikler

- **Tek URL modu** — tek sayfa
- **Manuel Liste modu** — birden fazla URL, satır satır
- **Otomatik kopyalama** — butona basınca prompt panoya kopyalanır
- **Markdown Yapıştır** — Claude çıktısını araca yükle, otomatik böler
- **ZIP İndir** — her sayfa ayrı `.md` dosyası olarak
- **İndir .md** — tek sayfayı indir
- **Markdown çıktı paneli** — token / karakter / satır istatistikleri
- **Kopyala / Düzenle / Temizle**
- **Geçmiş** — son 20 dönüşüm tarayıcıda saklanır
